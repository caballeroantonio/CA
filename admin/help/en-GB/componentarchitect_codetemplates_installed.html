Installed Code Templates Help

The following code templates will be pre-defined with the installation of Component Architect:

  • Joomla! 3.3 Standard
  • Joomla! 3.4 Standard

Do not change the records, plugins or files of a pre-defined Code Template!

For the most up-to-date information on the pre-defined code templates it is best to visit the Component Architect website (opens a new window).

Joomla! 3.3 Standard

The Standard Code Template for Joomla! 3.3 extended the functionality provided in the discontinued Joomla! 3.2 Standard code template by including the additional features in the Joomla! 3.x releases and meeting some revised class names and coding standards that were also introduced :

Additional functionality includes:

  • Support for Microdata
  • Improved routing performance

For the the full list of features supported by Joomla! 3.3 Standard code template see the Features Comparison tables (opens in new window).

Joomla! 3.4 Standard

The Standard Code Template for Joomla! 3.4extends the functionality provided in the Joomla! 3.3 Standard code template by including the additional features in the Joomla! 3.x releases and meeting some revised class names and coding standards that were also introduced.

For the the full list of features supported by Joomla! 3.4 Standard code template see the Features Comparison tables (opens in new window).
